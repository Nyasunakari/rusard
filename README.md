## Commandes  
`/gryffondor donne 100`  
Cela donne 100 points à gryffondor
  
`/serdaigle prend 100`  
Cela prend 100 points à Serdaigle, les points ne peuvent pas être négatifs.
  
`/poufsoufle set 100`  
Cela placerait les points de Poufsouffle à exactement cent.

`/serpentard add 50`  
Pareil que
`/s add 50`
